Documentation rapide

1. Inscription

Visitez la page d'inscription et remplissez tous les champs requis pour créer un compte.

2. Connexion

Entrez votre email et mot de passe pour accéder à votre compte.

3. Rôles

  • Admin : peut consulter la liste des inscrits, exporter PDF/Excel, gérer les événements.
  • Visiteur : peut consulter les événements et s’inscrire.

4. Export des données

L’admin peut exporter la liste des utilisateurs en format PDF ou Excel depuis le tableau de bord.
